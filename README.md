# Spletni sistem za vodenje in spremljanje zasedenosti fitnesa

## Navodila za zagon Python kode
### 1. Potrebne knjižnice
Za delovanje aplikacije so potrebne naslednje knjižnice:

- `tinydb`
- `flask`
- `requests`
- `itsdangerous`

Knjižnice se lahko namesti z ukazom:

pip install -r requirements.txt

### 3. zagon aplikacije

Po tem, ko so vse knjižnice nameščene, se lahko aplikacijo zažene z ukazom:

py main.py


### 4. Preveri aplikacijo v brskalniku

Aplikacija bo dostopna na naslovu:

```
http://127.0.0.1:5000
https://zakljucna-svuk.onrender.com
```
